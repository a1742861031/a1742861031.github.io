解题思路

采用滑动窗口的一个思路 sliding window

1、 创建两个指针，并且两个指针都指向字符串s的开头，创建一个Set，maxLength=0

2、通过指针 i 遍历字符串，当 set 中不含 s[i ]时，set中添加 s[i]，并且set长度与maxLength进行比较，实时更新maxLength的值

3、当set中含s[i]时，set中删除s[j]并且j右移，直到set中不含s[i]

4、 遍历字符串一遍后的结果即是最大长度
